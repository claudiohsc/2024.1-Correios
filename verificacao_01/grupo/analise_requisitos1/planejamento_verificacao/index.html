    Tabela 2 - Lista de verificação Aspectos éticos e termo de consentimento.

    ID Descrição Avaliação Referência Print
    1 No artefato desenvolvido eles citam evitar causar danos ou consequências negativas aos outros? tais como os perdas de informações, perdas de bens, danos a propriedades, ou impactos ambientais indesejados; respeitar a privacidade dos outros; e honrar a confidenciabilidade das informações; prejudicar ou causar danos a outras pessoas, seus bens, reputação um emprego. 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 140
    2 São levados em considerações os princípios de autonomia, beneficência, não maleficiência, justiça e equidade? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 140
    3 Informado os objetivos da pesquisa aos participantes? dizendo exatamente como vai ser a participação deles; informando o que vai ocorrer durante a coleta de dados; o tempo aproximado da coleta; os tipos de dados que serão coletados; e ainda como eles serão analisados. 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 141
    4 São informados a confiabilidade e a privacidade durante o tratamento dos dados? Tração do como isso será tratado na divulgação dos resultados? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 141
    5 É solicitado permissão para gravar voz ou imagem? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 141
    6 O consentimento livre e esclarecido dos participantes é garantidos através de um termo de consentimento? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 141
    7 É deixado claro para o participante o direito à Liberdade de se recusar a participar em qualquer momento ou fase da pesquisa? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 142
    8 São explicadas ou tratadas as formas de incentivo para participar das pesquisas do projeto? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 142
    9 No projeto é informado a validade e a confiabilidade dos dados coletados? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 143
    10 No projeto é informado a necessidade ou não de assegurar a confiabilidade dos dados dos sistemas apresentados ao participante? como um acordo de confidenciabilidade? 2. "7.4 Aspectos Éticos de Pesquisas Envolvendo Pessoas" página 143
